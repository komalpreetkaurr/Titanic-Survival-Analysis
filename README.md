# Titanic Survival Data Analysis

## Project Overview

This project analyzes the Titanic dataset to identify factors that influenced passenger survival during the Titanic disaster. Using Python and data analysis techniques, the dataset was explored to find meaningful insights about survival patterns.

## Objective

The objective of this project is to perform exploratory data analysis on the Titanic dataset and discover relationships between passenger characteristics such as gender, passenger class, and age with survival outcomes.

## Tools and Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab / Jupyter Notebook

## Dataset Information

The dataset contains information about passengers on the Titanic, including:

* Passenger class
* Gender
* Age
* Ticket fare
* Survival status

These variables were used to analyze patterns and identify factors that contributed to survival rates.

## Analysis Performed

The following analyses were conducted:

* Survival distribution of passengers
* Survival comparison by gender
* Survival comparison by passenger class
* Visualization of survival patterns using graphs

## Key Findings

* Female passengers had a significantly higher survival rate compared to male passengers.
* Passengers traveling in first class had better survival chances than those in second and third class.
* Younger passengers showed slightly higher survival rates.
